Набор программ для работы с PDF файлами.

PDFmerge - слияние PDF файлов. По-умолчанию ищет файлы в текущем каталоге удовлетворяющие признаку, на текущий момент это [u"(СБ)", u"(СП)", u"(Э3)"]. Сначала собираются вместе листы для каждого из этих признаков, а затем все в один файл.
